Exercise 2¶

A well established and diffused method for detecting a disease in blood fails to detect the presence of disease in 15% of the patients that actually have the disease.

A young UniPD startUp has developed an innovative method of screening. During the qualification phase, a random sample of n = 75 patients known to have the disease is screened using the new method.

a) What is the probability distribution of y, the number of times the new method fails to detect the disease ?¶

Answer:¶

The probability distribution of y will be a binomial distribution. If $p$ is the probability of the new method to correctly detect the disease in an ill person, we have $p(y) = \binom{n}{y} p^y (1-p)^{n-y}$.

b) On the n = 75 patients sample, the new method fails to detect the disease in y = 6 cases. What is the frequentist estimator of the failure probability of the new method ?¶

Answer:¶

An unbiased frequentist estimator for the parameter $p$ is $\hat{p}=\frac{y}{n}$. It is unbiased because:

$E[\hat{p}] = \frac{1}{n}E[y] = \frac{1}{n} \sum_{y} \binom{n}{y} p^y (1-p)^{n-y} y = \frac{1}{n}\frac{d}{dx}\sum_{y}\binom{n}{y} p^y (1-p)^{n-y}x^y |_{x=1} = \frac{1}{n}\frac{d}{dx}(1-p+px)^{n}|_{x=1} = p$.

Exercise 3¶

Lighthouse problem - but this time we don't know neither $\alpha$ nor $\beta$.

Solution¶

As done in class, we can derive the $p(x)$ from the $p(\theta)$, which is done through a change of variables. We get $p(x)dx = p(\theta)d\theta \implies \frac{1}{\pi}d\theta = p(x) |\frac{dx}{d\theta}|d\theta$. We have $x = \alpha + \beta \tan{\theta}$ which leads to $p(x) = \frac{1}{\pi\beta}\frac{1}{1+\left(\frac{x-\alpha}{\beta}\right)^2}$.

We can evaluate the posterior $p(\alpha,\beta|D) \propto p(D|\alpha,\beta)p(\alpha,\beta)$. To study this scenario, we can generate data having $\alpha_{true}=1\rm{km}, \beta_{true} = 1.5\rm{km}$. The prior we choose is uniform in the $[-2\rm{km}, 2\rm{km}] \times [1\rm{km}, 3\rm{km}]$ domain.

#Compute posterior for later timesteps and add rows to dataframe.
## To speed it up we only store the posterior at certain times.
counter <- 1
frames <- c(1,2,3,4,5,10,20,30,40,50,100,200,300,400,500,1000,5000,10000)
for(sample in samples){
    for(row in 1:nrow(grid)){
        for(col in 1:ncol(grid)){
            alpha <- 2 - (row-1)*0.1 
            beta <- 1 + (col-1)*0.1 
            grid[row,col] <- grid[row, col] * (1/(pi*beta)*1/(1+((sample-alpha)/(beta))^2)) #Update posterior
        }
    }
    
    #Normalization
    tot <- sum(grid*0.01)
    grid <- grid/tot
    
    #Store grid
    if(any(counter==frames)){
        for(row in 1:nrow(grid)){
            for(col in 1:ncol(grid)){
                alpha <- 2 - (row-1)*0.1
                beta <- 1 + (col-1)*0.1
                new_row <- c(beta, alpha, grid[row,col], counter)
                dataframe[nrow(dataframe)+1, ] <- new_row
            }
        }
    }
    counter <- counter + 1
}
